Navlight is a minimal, responsive, accessible, progressively-enhanced navigation widget for websites. I'm using it in the current responsive redesign of websites at my work. If you change the browser size while testing, you'll need to reload the page. I didn't put a listener on window size changes, because I wanted to maximize performance. If you like, you're obviously welcome to do so, though!

You can view this project at GitHub.
